# @manovaspace/tokens

Design tokens, semantic CSS variables, and font assets for applications using the manovaspace design system.

```bash
pnpm add @manovaspace/tokens
```

Also works with npm and yarn.

**Documentation:** [manovaspace.github.io/docs/design-system/packages/tokens](https://manovaspace.github.io/docs/design-system/packages/tokens/)

**Source:** [manovaspace/design-system](https://github.com/manovaspace/design-system/tree/main/packages/tokens)
